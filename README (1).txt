# Attendance Monitoring App

This app allows teachers to upload an Excel attendance sheet and enter a student's PRN or name to get a subject-wise attendance report with a bar chart.

## How to Run (on PC)

1. Install Python: https://www.python.org
2. Install required libraries:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py

## Features
- Excel file upload
- Search by PRN or name
- Subject-wise attendance
- Bar chart visualization
